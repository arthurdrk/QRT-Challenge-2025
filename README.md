# QRT-Challenge-2025

### Installation

Using `uv` is recommanded. First create a vitual environment with

```shell
uv venv
```

activate it with

```shell
./.venv/scripts/activate
```

then install the project with

```shell
uv pip install -e .
```

### Usage

You can run the different scripts using the command line interface `qrtdc`.
For more information, run

```shell
qrtdc --help
```
